##  ДЗ#2 Статичная верстка с использованием Tailwind (ветка:feature/static-layout)

Создать базовую статическую страницу каталога телевизоров. Никакой интерактивности — только вывод данных и вёрстка.  
Figma: https://www.figma.com/design/kqIZeoJ07t8ymMvjR0jLEM/Home-Assignment-2026?node-id=20-496&t=kPWWqMnOEayuhd1Z-0  

### Структура страницы
1. `<Header />` (sticky)
2. `<SideBar />` (фильтр + баннер)
3. `<Main>` (сетка товаров)
4. `<Footer />` (прижат к низу страницы)
